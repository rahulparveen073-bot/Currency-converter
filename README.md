# 💱 Currency Converter

A simple and user-friendly **Currency Converter** web application built with **Python** and **Streamlit**. It allows users to convert an amount from one currency to another using real-time exchange rates fetched from an online API.

---

## 🚀 Features

- 🌍 Convert between multiple international currencies
- 💹 Real-time exchange rates
- 💻 Clean and interactive Streamlit interface
- ⚡ Fast and lightweight application
- 🔄 Easy currency selection using dropdown menus
- 📱 Responsive and beginner-friendly UI

---

## 🛠️ Technologies Used

- Python 3
- Streamlit
- Requests
- Exchange Rate API

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/currency-converter.git
```

### 2. Navigate to the project folder

```bash
cd currency-converter
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```
